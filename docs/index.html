Before you begin

This is an Android class. To complete the coding exercises, you will need a Xamarin.Android development environment with either a simulator or a device to run the exercise on. Please see the Windows setup or macOS setup pages if you need help installing the Xamarin development environment.

Overview

In this class you will learn how to perform long-running operations on Android using Services. Services can be used to ensure that your apps remain responsive and can complete important long-running operations even when your application's UI isn't visible.

Objectives

  1. Create a Service
  2. Start a Service
  3. Elevate a service to foreground
  4. Bind to a Service

Lecture

AND210 - Android Services.

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Create a service
    Exercise
    In this exercise we will create and declare a new service.
  2. Start a service
    Exercise
    In this exercise we will add behavior to an existing service and start it from an Activity.
  3. Elevate a service to foreground
    Exercise
    In this exercise we will create a notification and use it to elevate a service to foreground.
  4. Bind a service
    Exercise
    In this exercise we will obtain a reference to a service by binding and use the reference to visualize data being obtained by the service.
